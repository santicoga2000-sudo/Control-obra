# JOIN · Control de Obra

Sistema de checklist visual para control y recepción de unidades habitacionales
(hotel / vivienda) en fase de acabados.

Permite:
- Crear zonas (habitaciones, áreas, sectores)
- Crear actividades por zona
- Marcar estado: PE / OBS / OK
- Registrar observaciones y fotos
- Visualizar avance ponderado por zona
- Mantener orden de actividades
- Eliminar actividades y zonas

## Estados
- **PE**: Pendiente
- **OBS**: Observaciones
- **OK**: Completado

## Uso
Abrir `index.html` en navegador (desktop o móvil).

En móviles, el botón Foto / Obs intentará abrir la cámara.

## Tecnología
- HTML
- CSS
- JavaScript (Vanilla)
- Persistencia local con localStorage
